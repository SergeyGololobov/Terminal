Linux terminal (GitBash) commands

1) Where I am (Посмотреть где я)           (pwd);
2) Create a folder (Создать папку)        (mkdir);
3) Go to the folder (Зайти в папку)       (cd namefolder/);
4) Create 3 folders (Создать 3 папки)     (mkdir -{app1,app2,app3} or mkdir folder1 folder2 folder3);
5) Go to any folder (Зайти в любоую папку)   (cd folder1);
6) Create 5 files (Создать 5 файлов) (3 txt, 2 json)   (touch file{1,2,3}.txt file{1,2}.json);
7) Create 3 folders (Создать 3 папки)       (mkdir -p {1,2,3});
8) List folder content (Вывести список содержимого папки)   (ls -l);
9) Open any txt file (+ Открыть любой txt файл)    (start namefile.txt);
10) + написать туда что-нибудь, любой текст.
11) + сохранить и выйти.
12) Exit the folder one level up (Выйти из папки на уровень выше)  (cd ..);
13) Переместить любые 3 файла, которые вы создали, в любую другую папку.  (mv file1.txt file2.txt file1.json folder1/);
15) Найти файл по имени (find file by name);
16) Просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
17) Вывести несколько первых строк из текстового файла (grep -A 5 root rm/Terminal.txt); 
18) Вывести несколько последних строк из текстового файла (grep -B 5 root /Terminal.txt);
19) Просмотреть содержимое длинного файла (команда less) изучите как она работает. (less Terminal.txt);
20) Вывести дату и время                     (date);

22) Редактировать содержимое файла qq.txt    (cat >> qq.txt);

27) Посмотреть содержимое папки qa_1 не заходя в нее (ls -la qa_1/);
28) Переместить файл из одной папки в другую (mv qa_1/q11.txt qa_3/q11.txt);
